# Artifacts

This directory contains locally reproducible outputs for the current scaffold.

- `verification_log.md`: smoke test execution summary.
- `policy_benchmark.csv`: simulator benchmark output.
- `all_reduce_plan.csv`: generated AllReduce communication plan.
- `all_gather_plan.csv`: generated AllGather communication plan.
- `reduce_scatter_plan.csv`: generated ReduceScatter communication plan.

These files are generated by:

```bash
./scripts/collect_artifacts.sh
```
